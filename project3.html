Introduction to LaTeX

LaTeX (pronounced "Lay-tech" or "Lah-tech") is a high-quality typesetting system commonly used for producing scientific and technical documents. It is based on the TeX typesetting system created by Donald Knuth. LaTeX provides a set of macros and commands that allow authors to focus on the content of their documents while the system handles the layout and formatting.

    LaTeX was developed in the early 1980s by Leslie Lamport as a set of macros for the TeX typesetting system. Here is a brief timeline of its development:
  • 1978: Donald Knuth creates TeX, a typesetting system designed to produce beautifully formatted books, particularly those with complex mathematical content.
  • 1984: Leslie Lamport releases LaTeX, which builds on TeX by providing a higher-level language with commands for common formatting tasks. LaTeX makes it easier for users to create complex documents without needing to delve into the more intricate aspects of TeX.
  • 1994: LaTeX2e is released as the standard version of LaTeX, providing a more stable and extensible framework.
  • 2000s-present: LaTeX continues to be maintained and extended by the LaTeX Project Team, with numerous packages developed by the community to add functionality.
Uses
    LaTeX is widely used in academia, especially in fields that require the inclusion of complex mathematical expressions, such as mathematics, physics, computer science, engineering, and economics. Its primary uses include:
  • Scientific Research Papers:LaTeX is the de facto standard for formatting research papers due to its ability to handle complex mathematical notations and its support for citation management.
  • Theses and Dissertations:Many universities provide LaTeX templates for students to ensure consistent formatting of theses and dissertations.
  • Books and Articles:Authors of technical books and articles use LaTeX to produce documents with professional-quality typesetting.
  • Presentations:With packages like Beamer, LaTeX can be used to create professional presentations.
  • Resumes and CVs:LaTeX can be used to create visually appealing and well-structured resumes and CVs.
  • Letters and Reports:LaTeX provides classes and templates for writing formal letters and reports.
Title and Author

The '\title{}' command sets the title of the document. The '\author{}' command sets the author of the document. The '\date{}' command sets the date of the document. '\date{\today}' automatically sets it to the current date. The '\maketitle' command generates the title, author, and date on the document.

\title{Sample LaTeX Document} \author{John Doe} \date{\today} \maketitle
Sections

The '\section{}' command creates a new section with the specified title. The '\subsection{}' command creates a subsection within a section. The '\subsubsection{}' command creates a sub-subsection within a subsection. The '\label{}' command sets a label that can be referenced elsewhere in the document using the '\ref{}' command.

\section{Introduction} This is the introduction section. Here, you introduce the topic and provide background information. \section{Methods} This section describes the methods used in the study. \subsection{Experimental Setup} Details about the experimental setup. \subsection{Data Collection} Explanation of how data was collected. \subsection{Data Analysis} Description of the data analysis techniques used.

Figures and Tables

Figure: The '\begin{figure}[h] ... \end{figure}' environment is used to include figures. The [h] option suggests the figure should appear "here" in the text. The '\includegraphics[width=\textwidth]{filename}' command is used to include an image file. The width=\textwidth option scales the image to the width of the text. The '\caption{}' command adds a caption to the figure. The '\label{}' command sets a label for the figure, which can be referenced later using the \ref{} command.

Table: The '\begin{table}[h] ... \end{table}' environment is used to create tables. The [h] option suggests the table should appear "here" in the text. The '\begin{tabular}{|c|c|c|} ... \end{tabular}' environment is used to create the actual table. The {|c|c|c|} argument specifies the number of columns and their alignment (centered in this case), with vertical lines between them. The \hline command creates a horizontal line across the table. Table cells are separated by the & symbol, and rows are ended with \\. The '\caption{}' command adds a caption to the table. The '\label{}' command sets a label for the table, which can be referenced later using the \ref{} command.

% in preamble \usepackage{graphicx} \begin{figure}[h] \centering \includegraphics[width=0.5\textwidth]{example-figure.png} \caption{Example figure caption.} \label{fig:example} \end{figure} \begin{table}[h] \centering \begin{tabular}{|c|c|c|} \hline Column 1 & Column 2 & Column 3 \\ \hline Data 1 & Data 2 & Data 3 \\ Data 4 & Data 5 & Data 6 \\ \hline \end{tabular} \caption{Example table caption.} \label{tab:example} \end{table}
Mathematics

Inline Math: Mathematical content can be included inline with text using the '$...$' environment.

Display Math: For larger equations or those that need to be centered, the '\begin{equation} ... \end{equation}' environment can be used. The '\[...\]' environment can also be used for unnumbered displayed equations.

\section{Inline Math} Inline math is used when mathematical expressions are included within a line of text. For example, the quadratic formula is given by $ax^2 + bx + c = 0$. Another example is the equation for the circumference of a circle, which is $C = 2 \pi r$ where $r$ is the radius. \section{Display Math} Display math is used for larger equations or those that need to be centered on their own line. These equations are usually more complex and benefit from being set apart from the main text. The quadratic formula can be displayed as follows: \begin{equation} x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a} \end{equation}
Hyperlinks

The '\usepackage{hyperref}' package is used to create hyperlinks within the document. The '\href{URL}{text}' command creates a hyperlink to an external URL with the specified anchor text.

\usepackage{hyperref} \href{https://www.overleaf.com/learn}{Overleaf LaTeX Tutorials}: Overleaf offers a comprehensive set of tutorials and examples for learning LaTeX.
Geometry

The '\usepackage{geometry}' package allows for easy configuration of page dimensions. The '\geometry{}' command is used to specify settings such as paper size and margins.

\usepackage{geometry} \geometry{a4paper, margin=1in}
